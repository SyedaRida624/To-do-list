# Flutter State & Persistence Dashboard

A high-fidelity Flutter application demonstrating the transition from pure UI to functional logic. This project focuses on reactive state management and local data persistence while maintaining a "Dark Mode" aesthetic.

## 🚀 Key Features

### 💎 Beautiful & Interactive UI
- **Animated Splash Screen**: Features an elastic scaling "Check" logo and a deep midnight mesh gradient.
- **Glassmorphic Dashboard**: Uses semi-transparent layers and refined borders for a modern, high-contrast look.
- **Micro-Interactions**: Custom button scaling and hover-style effects for a polished user experience.

### 🧠 State Management Basics
- **Manual Counter**: Real-time synchronization of  states using `setState`.
- **Reactive Task List**: Dynamic addition and removal of items with immediate UI feedback.
- **Optimized Rendering**: Implementation of `ListView.builder` for efficient memory usage.

### 💾 Persistent Data Storage
- **App Session Recovery**: Automatically retrieves saved counter values and tasks upon app restart.
- **SharedPreferences Integration**: Bridge between volatile app state and local device storage (`int` and `List<String>`).
- **Data Lifecycle**: Integrated "Save-on-Change" logic to ensure zero data loss during user interaction.
